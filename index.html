REDESIGN MAROON 5 SITE

Redesign of the existing website

DESCRIPTION

Maroon 5 has an existing website displaying information about the band, tours, online store, among other things. Overall the website has a good design, but some pages does not have any consistency, as for the grid and the layout.

The logo was a key part of the design process, the colors, and the form inspires me to create the layout using the red color on every page. Navigation bars are normally in horizontal layout, but for this website the vertical layout was more interesting to give a different look, modern and fresh.

In general, the redesign of the website has the same style, using the same fonts, colors and grid. One of the pages that were not part of the original site is the music and blog page. Both pages gave a plus to the site having a closer connection between artist-fans.

Smile with Flowers

Landing page design

DESCRIPTION

Smile with flowers is a fictional startup, which their main clients are brides, and bridesmaids. The whole ideas was to design a website that shows only a small portfolio and a contact form, with no online store for the moment.

Landing pages are becoming more popular; and for this specific project using a landing page was perfect to implement because of quantity of information. The colors were chosen to represent the beauty of the flowers. On the other hand the logo follows the concept of the landing page having a handmade style to represent softness.

Shine and Glow

Hair salon website design

DESCRIPTION

After looking at other websites, I noticed that hair salons usually do not try to establish a close relationship with customers; rather, they just show basic information like prices, locations, and promotions, among others.

Shine & Glow, started as an idea of a hair salon that connects with their customers, giving not only haircuts, but also experience to them. The website has the same concept, where a user that access the site can, look for some haircuts ideas, get appointment through the BOOK ONLINE page, and also they can still using the amazing products that the hair salon utilize by using the ONLINE STORE. The 4 columns grid was implemented in every page to have consistency, as well as the same style, colors and fonts.

Taco Magazine

Magazine layout design

DESCRIPTION

Easy Recipes

Magazine layout design

DESCRIPTION

The project involved creating a magazine cover and spread page from scratch.

The magazine used a three-columns layout grid on the spread pages, which helps the user to follow a hierarchy when reading. The cover page is very simple with a big image that covers almost 70% of the page to attract the eye of the customers. The articles shown in the cover page are around the image but these are still using the grid.

The idea was to show a clean design by organizing the elements in columns, grids and hierarchies, maintaining consistency throughout the whole design.
