La Casa de Leo

Rental House, Single Page Site

DESCRIPTION

Casa de Leo is a project for a rental house in Havana, Cuba. The owner was interested in having their website contain information about the rental house, and the city, including places to visit in the vicinity, as well as their contact information.

The final result consists of a single page website designed in Illustrator and developed with HTML5, CSS3, and Bootstrap. The colors represent part of the national identity of Cuba and combined with the shape on the home page creates an abstract Cuban flag. The layout for the website features a minimalist design, using only the necessary information to make the site friendly for the user.

Philips Museum Page

Museum Website

DESCRIPTION

Philips Museum started as a fictional museum, which allowed me to apply my skills as a designer and a web developer while creating sketches, wireframes, and coding the site using HTML5, CSS3, and Bootstrap.

The result consists of displaying only the home page and the exhibition page with information about and images of the museum. One of the most important parts of this project was to ensure the website could be viewed on multiple platforms including desktops, tablets and smart phones. The same layout, grid, fonts, and colors are used in both pages to create consistency throughout the website.

Juicy App

App design

DESCRIPTION

Nowadays, smartphones control everything. People are more inclined to check emails and blogs using smartphones rather than using desktops, which is why I chose to make a phone application.

As for my topic, being healthy, and eating more nutrients is a growing concern of the current generation. A great way to begin being healthy is to make your own juice. It is fast way to get nutrients, and it is highly portable which makes it easy to fit in to a busy schedule.

The application provides a quick reference for users on the benefits of fruits and vegetables, how to juice them, and what kinds go best together. It also provides a description of different juicers available to help users determine which fits their lifestyle best.In terms of design, the layout is simple and clean with repeating colors and fonts. It relies solely on vectors to create consistency on every page.

REDESIGN MAROON 5 SITE

Redesign of the existing website

DESCRIPTION

Maroon 5 has an existing website displaying information about the band, tours, online store. Overall, the website has a good design, but some pages lack consistency in the layout.

The logo was a key part of the design process, and the colors, inspired me to include the color red on every page. Navigation bars are normally in a horizontal layout; however, for this website the vertical layout provided more interest and gave it a different style.

The redesign website features many of the same styles, fonts, and colors from the original, but includes a music page and a blog page, which allow fans to connect with the artist.

Delta Airlines App

Redesign of the app located in the back part of the seats

DESCRIPTION

Delta Airlines has a distinctive design, using red and blue for almost every design that they propose. For this project, I redesigned the application to give it a different appearance for potential use on the screens located on the back part of the seats.

A flat design was a convenient way to provide a fresh look for the company. All the icons were created from scratch to create consistency in the stroke width, style and color. This application provides different services like music, online store, flight status, food and beverage, browser, and setting options. The background features the unique ability, to change in response to weather and the time of day.

Smile with Flowers

Landing page design

DESCRIPTION

Smile with flowers is a fictional startup company, whose main clients are brides and bridesmaids. The goal of the project was to design a website that shows only a small portfolio and a contact form, with no online store for the moment.

Shine and Glow

Hair salon website design

DESCRIPTION

After looking at various websites, I noticed that hair salons usually do not try to establish a close relationship with customers; instead, they just show basic information like prices, locations, and promotions.

Shine & Glow started as an idea for a hair salon that connects with their customers, providing not only a haircut, but also an experience. The website follows the same concept, allowing the user to access the site and look for haircuts ideas, book appointments online, and shop for products using the online store page. The website features a 4 column grid with similar styles, colors, and fonts, implemented on every page to create consistency.

Taco Magazine

Magazine layout design

DESCRIPTION

Taste is an existing magazine that constantly changes its logo and cover page layout based on a monthly concept. For this project, I decided to redesign the cover, content, and spread page using a Mexican food concept.

The content was organized with a three-grid layout, to create a clean spread that displayed information with images to complement the idea. The colors selected for the logo and pattern represent national identity, and are often used in handcrafts, clothes and mexican celebrations.

Easy Recipes

Magazine layout design

DESCRIPTION

The project involved creating a magazine cover and spread page from scratch based on an original idea.

The magazine used a three-column layout grid on the spread pages, which follow a hierarchy and make it easier for the user to read. The cover page is very simple with a big image that covers almost 70% of the page to attract customers but not overwhelm them visually. The articles displayed on the cover page surround the image following a grid pattern. The design used on the spread pages provides consistency and accessibility.
